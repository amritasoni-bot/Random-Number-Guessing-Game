# Random-Number-Guessing-Game
This is a beginner-friendly game that I made while learning Python before my college begins.

![Game Screenshot](Screenshot%202026-06-05%20122858.png) 

In this game I use Python's 'random' module where users guess the number and program provides hint by showing your number is too big/small as per generated number. It also has option to QUIT the game in mid.

## Through this project, I learned:
1-) Handling user's data
2-) Loops
3-) Python's random module
4-) Conditional Statements 

This programme is just a part of my journey of learning skills and programming language.
